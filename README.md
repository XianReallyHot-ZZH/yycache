# yycache
[yycache](https://github.com/XianReallyHot-ZZH/yycache)是一个基于springboot构建的缓存server。

## 简介
yycache实现了redis的协议，支持redis的部分指令，redis-cli可以正常访问，yycache相当于是基于springboot去mock了一个redis的server。
